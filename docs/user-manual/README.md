## 快速开始（30 秒）

1. 打开 `http://<部署地址>:8900`
2. 用管理员提供的账号登录
3. 顶栏可切换**中文 / English**，浏览器会自动记住你的选择
4. 看头像右侧的**橙色标签**——那是你的角色；翻到 [01 角色与权限](./01-roles-and-permissions.md) 的对应章节开始

## 术语速查

| 缩写 | 中文 | 含义 |
|:---|:---|:---|
| PR | 采购申请 | Purchase Requisition，采购的起点 |
| PO | 采购订单 | Purchase Order，正式下单给供应商 |
| 三单匹配 | 三单匹配 | PO / 收货 / 发票 三方数据核对 |
| SSO | 单点登录 | Single Sign-On（当前为骨架，生产环境将启用 ADFS） |
| SKU | 标准物料编号 | Stock Keeping Unit |
