# 欢迎使用 Mica（觅采）

**Mica（觅采）** 是你们公司的采购管理系统。无论你是提交采购申请的 IT 同事、审批订单的部门经理、还是核对付款的财务伙伴——这本手册都能帮你快速上手。

---

## 30 秒快速开始

1. 打开浏览器，访问公司提供的 Mica 地址
2. 输入管理员分配的**用户名和密码**登录
3. 看右上角**角色标签**——它决定你能做什么
4. 跳到 [各角色工作流](03-per-role-workflow.md) 找到你的角色，按图操作即可

!!! tip "小技巧"
    右上角可以切换 **中文 / English**，也可以切换 **浅色 / 暗色主题**。你的选择会被自动记住。

## 常用术语

| 缩写 | 含义 |
|:---|:---|
| **PR** | 采购申请（Purchase Requisition）—— 采购的起点 |
| **PO** | 采购订单（Purchase Order）—— 正式下单给供应商 |
| **SKU** | 标准物料编号 —— 统一的物料命名和价格跟踪 |
| **三单匹配** | 采购订单 + 收货记录 + 发票 三方数据核对 |
