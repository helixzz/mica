# 管理员指南导读

本组文档面向 **系统管理员 / 运维负责人 / 平台维护者**，用于说明如何部署、配置、运维和排查 Mica。

## 建议阅读顺序

1. [部署与运维](01-deployment-and-ops.md)
2. [HTTPS / TLS](02-https-and-tls.md)
3. [系统管理控制台](03-system-console.md)
4. [SAML SSO](04-saml-sso.md)
5. [权限架构](05-permissions.md)
6. [备份与恢复](06-backup-restore.md)
7. [故障排查与升级](07-troubleshooting-and-upgrade.md)
8. [飞书集成](08-feishu-integration.md)

## 本指南覆盖什么

- 首次部署与日常运维脚本
- 生产环境 HTTPS / TLS 启用与证书轮换
- 系统参数、用户、分类、LLM 模型与审计日志管理
- ADFS / Microsoft Entra ID 的 SAML SSO 配置
- 权限架构、Cerbos 策略与合规审计
- 备份恢复、故障排查与版本升级
- 飞书集成（消息通知 + 付款审批工作流）

如果你只是普通业务用户，请返回 [首页](../README.md) 或用户手册部分。
