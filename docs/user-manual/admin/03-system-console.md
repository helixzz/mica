# 系统管理控制台

管理员登录后，可从左侧导航进入 **系统管理**。

## 系统参数

可在线维护运行时参数，例如：

- 审批金额阈值
- JWT 有效期
- SKU 基准价窗口天数
- 合同到期提醒天数
- 上传文件大小限制
- 默认付款到期天数

## 分类与主数据

可维护的核心分类包括：

- 成本中心
- 开支类型
- 采购分类

这些维度会被采购申请、订单、物料与报表广泛引用。

## LLM 模型配置

可配置 OpenAI 兼容模型服务，例如 DeepSeek、GLM、Kimi、通义等，并将具体功能路由到指定模型。

## 用户管理

管理员可以：

- 创建或停用用户
- 分配角色
- 管理密码与登录方式

## 审计日志

关键操作会被记录到审计日志中，用于排查与合规追踪。
