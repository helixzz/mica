# System Console

Administrators can open **System Administration** from the application navigation menu.

## System parameters

Runtime settings can be changed online, including:

- approval amount thresholds
- JWT lifetime
- SKU benchmark window length
- contract expiry reminder days
- upload size limits
- default payment due-day settings

## Classifications and master data

Core maintained dimensions include:

- cost centers
- expense types
- procurement categories

## LLM model configuration

Mica supports OpenAI-compatible model providers such as DeepSeek, GLM, Kimi, and Tongyi. Administrators can register models and route AI features to the correct backend model.

## User management

Administrators can create users, disable them, manage roles, and adjust sign-in options.

## Audit logs

Critical business and administrative actions are captured for traceability and compliance review.
